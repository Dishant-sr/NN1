# MNIST Neural Network (From Scratch)

- Built using NumPy only
- No TensorFlow / PyTorch
- Use Activation fucntions - ReLU & Softmax
- Implements:
  - Forward propagation
  - Backpropagation
  - Gradient descent

Accuracy: ~88% (improving)
